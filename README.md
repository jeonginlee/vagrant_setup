# vagrant_setup
Set up config files for Vagrant + venv 


### Includes:
* git and repo setup (URL in first command line argument)
* vim, .vimrc, and Vundle
* tmux


## To run:
```
sh run.sh <URL FOR EMPTY REPO>
```

### Additional set-up:
1. :PluginInstall (inside of vim)
